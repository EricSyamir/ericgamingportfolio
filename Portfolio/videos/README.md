Place your project preview videos in this folder. These videos will display when users hover over project cards.

8. `fullstack-project.mp4` - Full-Stack Web Apps

## Video Specifications

**Recommended Settings:**
- **Duration**: 5-10 seconds (short preview)
- **Format**: MP4 with H.264 codec
- **Resolution**: 1280x720 (HD) or 1920x1080 (Full HD)
- **File Size**: Under 5MB per video for optimal loading
- **Aspect Ratio**: 16:9

## Tips for Creating Great Preview Videos

1. **Show Key Features**: Capture the most impressive functionality
2. **Keep it Short**: Users hover briefly - make every second count
3. **No Audio Needed**: Videos play muted by default
4. **Good Lighting**: Ensure clear visibility of the interface
5. **Smooth Transitions**: Avoid jerky movements

## Video Compression Tools

- **HandBrake** (Free, cross-platform): https://handbrake.fr/
- **FFmpeg** (Command-line): `ffmpeg -i input.mp4 -vcodec h264 -acodec aac output.mp4`
- **Online Converters**: CloudConvert, Online-Convert

## What if I Don't Have Videos Yet?

The website includes fallback displays that show "Preview Unavailable" if videos don't load. You can:
1. Add placeholder images instead
2. Use screenshots or GIFs converted to MP4
3. Add videos later without breaking the site
