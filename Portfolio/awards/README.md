# Award Images

## Image Guidelines
- Use high-quality, professional images
- Recommended size: 600x400 pixels
- Supported formats: JPG, PNG

## Required Images
1. `oh-my-code.jpg` - Oh My Code Competition Champion
2. `proton-dx.jpg` - Proton DX Hackathon Award
3. `hacking-weekend.jpg` - Hacking Weekend Hackathon Award
4. `yutp-scholarship.jpg` - YUTP MARA Scholarship

## Placeholder Images
If you don't have actual award images, you can:
- Use team/event photos
- Use screenshots from the competition
- Create simple graphic representations

## Recommended Tools for Image Preparation
- Adobe Photoshop
- GIMP (Free)
- Canva
- Pixlr (Online)

## Image Optimization
- Compress images to reduce file size
- Maintain good resolution
- Use consistent styling across images
